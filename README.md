#  Student Performance Predictor

##  Problem Statement

Students often struggle to understand how their study habits affect their academic performance.

##  Objective

To build a Machine Learning model that predicts student marks based on study hours and attendance.

##  Technology Used

* Python
* Pandas
* Scikit-learn
* Matplotlib

##  How It Works

* Takes input: Study Hours & Attendance
* Uses Linear Regression
* Predicts expected marks
* Displays performance level

##  How to Run

1. Install required libraries:
   pip install pandas scikit-learn matplotlib

2. Run the program:
   python main.py

3. Enter:

   * Study Hours
   * Attendance

##  Output

* Predicted Marks
* Performance Category
* Graph Visualization
